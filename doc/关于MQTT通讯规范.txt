# 音量控制主题示例

音量控制默认订阅 `webrtcsettings.json` 中 `mqtt_sound.topic` 配置的主题（例如 `LHTDA4B48SY9JA047/sound`）。

消息体支持以下两种格式，值会被限制在 0~100：

1. 直接发送数字文本：
```
45
```

2. 发送包含 `volume` 字段的 JSON（数字或字符串皆可）：
```
{"volume":80}
```
或
```
{"volume":"65"}
```

空白或无法解析的载荷会被忽略，并在日志中提示解析失败。
